# SafeSpace | Anonymous Student Wellness Platform

SafeSpace is a digital wellness support system designed to provide students with a secure and anonymous environment to access mental health resources, support systems, and community guidance.

## 🚀 Overview
Being a student can be overwhelming. SafeSpace bridges the gap between students and wellness support by utilizing a secure architecture that prioritizes user privacy and ease of access.

## ✨ Key Features
* **Complete Anonymity:** High-level privacy for students seeking support.
* **Wellness Support System:** Built-in resources for mental and emotional well-being.
* **Secure Architecture:** Developed with modern web principles to ensure data integrity.
* **Responsive Design:** Accessible across various devices for on-the-go support.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, TypeScript (React/Vite)
* **Development Environment:** Visual Studio Code
* **Version Control:** Git & GitHub
